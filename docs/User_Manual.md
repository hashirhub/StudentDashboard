# Student Dashboard User Manual

## Login

1. Open the application.
2. Go to Student Login.
3. Use one of the demo student emails:
   - irtaza@studentdash.local
   - sufyan@studentdash.local
   - hashir@studentdash.local
4. Click **Open Student Dashboard**.

## Student Workflows

The student can:

- View dashboard overview
- View profile
- Update phone/contact number
- View enrolled courses
- Search courses
- View attendance
- View grades
- View assignments
- View fee invoices
- Pay/mark pending fee as paid
- Read notifications
- Mark notifications as read
- View events
- Create support tickets
- Track own support tickets

## Troubleshooting

If the app does not run, make sure Visual Studio is using the StudentDashboard.Web profile and the URL is HTTP: `http://localhost:5234`.
