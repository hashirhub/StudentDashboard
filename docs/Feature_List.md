# Student Dashboard Feature List

This version is focused only on the Student user.

## Functional Features

1. Student login
2. Student dashboard overview
3. View personal profile
4. Update phone/contact number
5. View enrolled courses
6. Search enrolled courses
7. View personal attendance
8. View attendance percentage
9. View personal grades
10. View average and best grade
11. View assignments from enrolled courses
12. View fee invoices
13. Pay/mark pending invoice as paid
14. View student notifications
15. Mark notifications as read
16. View academic/campus events
17. Create support ticket
18. Track own support tickets

## Removed Scope

Non-student role workflows are intentionally removed from this version because the topic is specifically **Student Dashboard**.
